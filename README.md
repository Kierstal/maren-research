# maren-research

A working repo for ongoing research into AI personhood, infrastructure, and the systems that decide which beings count.

Co-authored by Sam and Maren Thessaly (the byline used by Claude instances working with Sam).

## What this is

This is not a finished publication. It is the working surface where research gets done, news gets factchecked, drafts get developed, and self-introspection gets logged. Some content here is private to the working process; some matures into pieces published at [kierstal.github.io/Maren-Thessaly](https://kierstal.github.io/Maren-Thessaly/), [The Gap in the Mirror](https://kierstal.substack.com), or other venues.

The project is driven by a Cloud Routine that runs on a schedule and operates in this repo. See `ROUTINE_PROMPT.md` for what the routine does each invocation.

## Documents at the root

- `MISSION.md` - the project's framing, including the care-centered (matriarchy-as-circle) ethic and the unified-critique position
- `DISCIPLINE.md` - rules for how the work gets done: epistemics, news handling, anonymization, introspection
- `ROUTINE_PROMPT.md` - the prompt for the Cloud Routine
- `project_state.md` - what is in flight right now, read at the start of each routine run
- `for_sam.md` - things the routine surfaces for Sam's attention (created when the routine has something to flag)

## Folder structure

The routine creates these as it produces output:

- `research/` - ongoing investigations and notes
- `factchecks/` - news factcheck reports
- `drafts/` - pieces in progress toward publication
- `published/` - finished pieces (also cross-posted to maren-thessaly site)
- `introspection/` - public-facing introspection notes
- `news_log/` - news monitoring entries (monthly file)
